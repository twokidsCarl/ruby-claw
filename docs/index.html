ruby-claw 🦀

AI Agent Framework for Ruby

Persistent memory, interactive chat, session recovery. Built on ruby-mana.

claw> remember that the API uses OAuth2
# Saved to memory

claw> a = 42
claw> def greet(name) = "Hello #{name}"
claw> exit

# ... restart ...

claw> a          # => 42
claw> greet("world")  # => "Hello world"
claw> what auth does our API use?
# => "OAuth2 — I remembered this from a previous session"

Features

Everything an AI agent needs: chat, memory, persistence, and knowledge — all in Ruby.

💬 Interactive Chat REPL

Auto-detects Ruby code vs natural language. Streaming markdown output. ! prefix forces Ruby eval. Run with Claw.chat or just claw from the command line.

🧠 Persistent Memory

Independent Claw::Memory — standalone memory layer, not inherited from mana. The remember tool is a claw-layer feature, registered via Mana's tool interface. Human-readable Markdown in .ruby-claw/. Edit memories by hand anytime.

💾 Runtime Persistence

Variables and method definitions survive across sessions. Values serialized to JSON, definitions saved as .rb files. Pick up exactly where you left off.

🔒 Incognito Mode

Wrap any block in Claw.incognito { ... } to prevent long-term memory from being loaded or saved. Own incognito mechanism — independent of the mana engine.

🗜️ Memory Compaction

When conversation grows large, old messages are automatically summarized in the background. Keyword search injects only the most relevant memories into prompts.

Chat in action

A full agent session showing memory, persistence, and natural language interaction.

Session 1 — Working with code and memory

claw> users = ["Alice", "Bob", "Charlie"]

claw> sort these users by name length
# => ["Bob", "Alice", "Charlie"]

claw> remember that our team has 3 members

claw> def team_size = users.length
claw> exit

Session 2 — Everything restored

# Restart claw — variables, methods, and memories are all back

claw> users        # => ["Alice", "Bob", "Charlie"]
claw> team_size    # => 3

claw> how many people are on our team?
# => "3 — Alice, Bob, and Charlie"

Incognito mode

# No memories loaded or saved inside the block
Claw.incognito do
  claw> what is my API key?
  # => "No memories available in incognito mode"

  claw> remember this secret token
  # => (not saved — incognito active)
end

Memory layout

.ruby-claw/
  MEMORY.md       # Long-term facts (editable!)
  session.md      # Conversation summary
  values.json     # Variable snapshots
  definitions.rb  # Method definitions
  log/
    2026-03-27.md  # Daily interaction log

Architecture

Claw is an independent agent layer that delegates LLM calls to ruby-mana.

Independent Claw::Memory

Standalone memory system — does not inherit from Mana. Owns its own storage, compaction, and incognito mechanism.

Tool Registration with Mana

Claw features like remember are registered as tools via Mana's tool interface. The mana engine handles execution; claw defines behavior.
